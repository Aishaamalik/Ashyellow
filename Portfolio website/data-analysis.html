Predictive modeling offers insights into factors influencing help-seeking but faces challenges due to weak correlations. Insurance Claim Fraud Detection

This project focused on detecting fraudulent insurance claims using machine learning techniques. I approached the problem by applying supervised classification, with the target variable being whether a claim was fraudulent or not. After preprocessing the data, I evaluated multiple machine learning models, including Logistic Regression, Random Forest, XGBoost, LightGBM, and Isolation Forest. The LightGBM model emerged as the best performer, achieving 75% accuracy and a 0.82 ROC-AUC score, along with a 40% recall for fraud cases. The challenges of class imbalance were addressed through techniques like stratified sampling and probability thresholding. The model can significantly aid insurance companies in detecting fraudulent claims early, reducing financial losses and operational inefficiencies. Future improvements include adding more features and exploring advanced anomaly detection techniques.
