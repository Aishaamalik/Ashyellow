Web Apps & Dashboards Projects

Here are some of my web application and dashboard projects, showcasing my skills in building interactive, data-driven solutions for business and research. These projects leverage technologies such as React, Flask, Django, and RESTful APIs to deliver real-time insights and user-friendly interfaces.

Crypto Portfolio Tracker

I developed the Crypto Portfolio Tracker web app using React.js for the frontend, ensuring a responsive UI with TailwindCSS and live data visualizations via Chart.js. The backend, powered by FastAPI, handles real-time price updates and forecasting with Prophet for 7-day predictions. I integrated Binance API for fetching coin data and implemented WebSockets for live updates. The app features a dark/light theme toggle and provides users with a clean, interactive way to track and predict cryptocurrency trends.

Smart Health Dashboard

For the Smart Health Dashboard, I built a web app using React.js for a responsive interface, enhanced with TailwindCSS for styling and Chart.js for visualizing health data like heart rate, steps, and sleep patterns. The backend, developed with FastAPI, processes uploaded CSV/JSON health data and uses Scikit-learn for activity predictions and anomaly detection. Users can view trends, receive health predictions, and get alerts for unusual data changes. I implemented a clean, user-friendly design with a dark/light theme toggle and consistent layout elements.

AI-Powered Resume Analyzer

For the AI-Powered Resume Analyzer, I developed a web app that uses AI and Natural Language Processing (NLP) to analyze resumes. The frontend, built with React.js, allows users to upload their resumes, while the backend, powered by FastAPI, processes and extracts key information such as skills, experience, and qualifications. The AI model evaluates the resume against job descriptions to give feedback and recommendations for improvement. The app provides users with actionable insights to optimize their resumes for better job opportunities.

Climate & Air Quality Visulizer

For the Climate Visualizer project, I developed a full-stack web application to explore, analyze, and visualize global air quality data. The backend, built with Flask, serves air quality data and advanced analytics via a REST API, processing data from a CSV file. The frontend, developed using React.js and TailwindCSS, provides interactive dashboards, charts, and maps for users to explore trends, pollutants, and forecasts. Key features include forecasting with scenario analysis, pollutant composition, and comparative analysis across cities and countries. The app supports real-time data updates, anomaly detection, and interactive visualizations, making it an invaluable tool for air quality research and public awareness.
